## Установка

Обязательно установите зависимости:

```bash
# npm
npm install

```

## Добавление .env

Добавьте файл .env с содержимым:

```bash

TOKEN=your_api_token_open_weather_map_api

```

## Запуск сервера для разработки

Запустите сервер разработки на `http://localhost:3000`:

```bash
# npm
npm run dev
```
